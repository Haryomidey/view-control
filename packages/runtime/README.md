# ViewControl Runtime

Install with npm:

```js
import { init } from '@viewcontrol/runtime';

init({
  projectId: 'vc_your_project_key',
  apiUrl: 'https://api.your-domain.com'
});
```

Or use the CDN:

```html
<script src="https://api.your-domain.com/cdn/viewcontrol.js" data-project-id="vc_your_project_key" async></script>
```
